# Metric Definitions

Single source of truth for all metrics used by the dashboard and the AI chat.
Defined in code at `src/metrics/definitions.py`.

| Metric ID | Description |
|---|---|
| `total_revenue` | Sum of all revenue |
| `total_orders` | Total number of orders (rows) |
| `avg_order_value` | Average revenue per order |
| `top_products_by_revenue` | Top 10 products ranked by total revenue |
| `revenue_by_region` | Total revenue broken down by region |
| `top_products_by_quantity` | Top 5 products ranked by units sold |
| `revenue_by_channel` | Total revenue broken down by sales channel |
